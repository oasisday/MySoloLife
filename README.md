-[] 번역에서 사진이 번역되는건지 모르겠습니다. 시간이 5분이 지나도 안 됐습니다.
-[x] 홈 화면에서 카메라를 누르면 정상적으로 작동이 되지 않습니다.

-[] 카카오 로그인 유저는 이메일 추가가 불가능합니다.
-[x] 이메일 친구 추가 버튼이 작동하지 않습니다.
-[] 알람이 뜨고 그 알람을 통해 녹음을 진행할 때 녹음을 취소하면 팅기는 오류가 있습니다.
-[] 이따금씩 발생하는 튕김 현상 개선 필요합니다.
